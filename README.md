# Retail Inventory Analytics with Microsoft Fabric

Built an end-to-end analytics solution using Microsoft Fabric. The project ingests retail sales data through Dataflow Gen2, stores it in a Fabric Warehouse, performs SQL-based analysis, and creates business metrics using Semantic Models and DAX measures.

## Key Learnings
- Data ingestion and transformation with Dataflow Gen2
- Warehouse creation and SQL querying in Microsoft Fabric
- Semantic Modeling and DAX measure creation
- Data quality analysis (duplicate detection and handling)
- GitHub Actions, YAML workflows, and CI/CD fundamentals

## Architecture
CSV → Dataflow Gen2 → Fabric Warehouse → Semantic Model → Power BI Report
